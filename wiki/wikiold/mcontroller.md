# MController documentation
## Interfaces
Представляет собой словарь, где ключом является его номер, а значеием - сокет для чтения/записи
- При отправке пакета в начало добалявяется байт с номером iid - внутренний id (0 если не используется)
- По стандарту контроллер с помощью select (interfaces.values()) достает пакет, обрабатывает его, генерирует/обновляет пакет и отправляет его с помощью роутера на нужный интерфейс

## Communicators
Представляет собой словарь, где ключом является внутренний адрес (uint16, uint16), а значнием - пара (interface_id, iid)

## Router
Класс роутера, описан в [router](/wiki/router.md). На вход получает адрес назначения и возвращает внутренний адрес.

## Meta
Словарь с данными контроллера. Хранит:
- local_addresses: массив с собственными адресами (в том числе loopback)
